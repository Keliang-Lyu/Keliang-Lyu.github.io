解决 Github Pages 网页无法打开的问题

前段时间发现自己都无法访问 github.io 的页面，之前一直以为是被封了的原因。
今天找到了个解决方法，可以解决无法打开 Github Pages 的问题。

进入控制面板 -> 网络和 Internet -> 网络共享中心 选择联网的活动网络

选择属性 -> 网络 -> Internet协议版本4（TCP/IPv4）-> 属性 -> 修改DNS：8.8.4.4 备用：1.1.1.1

就可以打开 github.io 了

但是修改了之后必应搜索用不了了

参考微软必应国内版打不开解决方法的方法，通过修改 host 解决（C:\Windows\System32\drivers\etc）
